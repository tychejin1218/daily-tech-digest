## 2026-04-24 주요 IT 뉴스 3선

### 1. OpenAI, GPT-5.5 공개 — '에이전틱 컴퓨팅' 본격 시동
OpenAI가 4월 23일 GPT-5.4 출시 후 불과 몇 주 만에 GPT-5.5를 공개했다. 코딩·컴퓨터 조작·장기 리서치 능력을 크게 끌어올렸으며, Greg Brockman 사장은 "새로운 지능의 클래스이자 더 에이전틱·직관적 컴퓨팅으로의 큰 걸음"이라고 표현했다. Plus·Pro·Business·Enterprise 구독자 대상 ChatGPT 및 Codex에 즉시 롤아웃됐다.

> 💡 **왜 중요한가**: LLM API 가격·성능 경쟁이 분기 단위에서 주(週) 단위로 좁혀지고 있어, 사내 코딩 에이전트·리서치 파이프라인을 설계할 때 모델 스왑을 전제로 한 추상화 계층이 사실상 필수가 됐다.

---

### 2. Google, TPU v8을 훈련용 'Sunfish'·추론용 'Zebrafish'로 분할
Google Cloud Next 2026에서 공개된 8세대 TPU는 Broadcom이 제작하는 훈련 특화 'Sunfish(TPU v8ax)'와 MediaTek이 설계하는 추론 특화 'Zebrafish(TPU v8x)'의 이원 전략으로 출시된다. 훈련용은 Ironwood 대비 2.8배 가격 대비 성능, 팟당 121 엑사플롭스를 제공하며, 추론용은 288GB HBM + 384MB 온칩 SRAM으로 에이전트 워킹셋을 칩 내부에 고정해 달러당 성능을 80% 개선한다.

> 💡 **왜 중요한가**: AI 에이전트 워크로드에서 추론 비용이 지배적 원가가 되면서 하이퍼스케일러가 칩 공급망까지 이원화·경쟁 구도로 재편하고 있다. Vertex AI·Gemini API 가격이 추가로 떨어질 여지가 크다.

---

### 3. Google, 'Gemini Enterprise Agent Platform' 출시 — Agent Designer 프리뷰 포함
Vertex AI·Agent Studio·Model Garden(200+ 모델)을 단일 엔드투엔드 플랫폼으로 묶고, Agent Designer(노코드 비주얼 플로우 캔버스), Agent Inbox(에이전트 활동 관리), Skills, Projects, 장기 실행 에이전트, 그리고 각 에이전트에 고유 Identity를 부여하는 보안 레이어까지 한 번에 제공한다. Wiz 통합 기반 Agentic Defense, 크로스클라우드 Lakehouse·Knowledge Catalog 기반 Agentic Data Cloud도 동시 공개됐다.

> 💡 **왜 중요한가**: "에이전트 빌더 → 오케스트레이터 → 거버넌스"를 분리해 조립하던 엔터프라이즈 AI 스택이 벤더 통합 플랫폼으로 수렴하는 신호로, 자체 에이전트 인프라를 구축 중인 팀은 플랫폼 락인 vs. 개발 속도의 재평가가 필요하다.
