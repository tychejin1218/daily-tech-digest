# 오늘의 IT 뉴스 (2026-04-21)

### OpenAI Agents SDK 대규모 업데이트 — Shell·apply_patch 툴과 샌드박스 런타임 네이티브 탑재
OpenAI가 4월 15일 Agents SDK를 전면 업데이트하며 파일을 직접 편집하고(apply_patch), 셸 명령을 실행하며(shell tool), 장기 태스크를 안전히 돌릴 수 있는 **네이티브 샌드박스 실행 환경**과 **구성 가능한 메모리·컨텍스트 컴팩션·재사용 가능한 에이전트 스킬**까지 표준 프리미티브로 포함했다. MCP 통합, AGENTS.md 기반 커스텀 지시, Codex 스타일 파일시스템 툴이 기본 하네스에 들어가면서 "프레임워크 조립" 없이도 프로덕션급 코딩 에이전트를 만들 수 있게 됐다. 우선 Python부터 GA, TypeScript는 후속 지원 예정.

> 💡 **왜 중요한가**: 그동안 LangGraph·CrewAI 같은 서드파티 프레임워크가 담당하던 "에이전트 런타임" 영역을 모델 제조사가 직접 표준화한 신호 — LLM API를 쓰는 팀은 더 이상 하네스를 자체 구현할 필요가 없다.

### Anthropic Claude Opus 4.7, Google Cloud Vertex AI Model Garden에서 정식 GA
4월 10일 Google Cloud는 Claude Opus 4.7을 Vertex AI Model Garden에서 **정식(GA)** 출시했다. Opus 4.6 대비 모호한 지시 처리·문제 해결 철저성·지시 준수가 개선됐고, 고해상도 이미지 입력과 장기 태스크용 확장 메모리도 들어갔다. **1M 토큰 컨텍스트·글로벌 엔드포인트·유연한 TTL 프롬프트 캐싱·배치 예측·웹 검색**이 지원되며, 리타이어먼트는 빨라야 2027년 4월 16일이다.

> 💡 **왜 중요한가**: 멀티클라우드 규제·데이터 거주지 제약이 있는 국내 대기업·공공에서 "Anthropic 직접 계약 없이도 Opus 4.7을 GCP 리전 안에서" 호출할 수 있게 됐다 — 에이전트·코드 도구 아키텍처 선택지가 늘어나는 즉각적 효과.

### DeepSeek V4, 4월 말 임박 — Huawei Ascend 950PR에서 돌아가는 첫 프런티어 모델
트레이더·업계 리크에 따르면 DeepSeek V4가 **4월 마지막 2주 안에 공개**될 가능성이 높다. 스펙은 1조 파라미터 MoE(토큰당 활성 ~37B), 1M 컨텍스트(Engram 조건부 메모리), 텍스트·이미지·비디오 네이티브 멀티모달이며 **가중치는 Apache 2.0**으로 풀릴 전망. 결정적 차이점은 실행 인프라 — NVIDIA/AMD가 아니라 **Huawei Ascend 950PR**에서 돌아가도록 학습된 최초의 프런티어 모델이다.

> 💡 **왜 중요한가**: "오픈 가중치 + 비-NVIDIA 스택"이 동시에 프런티어급 성능에 도달하는 첫 사례가 될 수 있어, 국내 기업의 AI 서빙 GPU 공급망·벤더 락인 전략 재검토를 촉발할 가능성이 크다.
