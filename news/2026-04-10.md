### Instant 1.0 정식 출시 — AI 코딩 앱용 오픈소스 백엔드 공개

4년간 개발된 Instant가 1.0 버전을 오픈소스로 공개했다. Postgres 위에 멀티테넌트 DB와 Clojure 기반 싱크 엔진을 얹어, 프로젝트마다 VM을 띄우지 않고 DB 행(row) 삽입만으로 순간 프로비저닝한다. 비활성 앱은 컴퓨트·메모리 비용이 0에 수렴하면서도 요청이 오면 즉시 응답한다.

> 💡 **왜 중요한가**: "콜드 스타트 30초" 문제를 구조적으로 회피하는 BaaS 설계로, AI 에이전트가 생성하는 다량의 소형 앱 시대에 적합한 새 표준을 제시한다.

### AWS Interconnect 멀티클라우드 프리뷰 — Google Cloud와 전용 회선 직결

AWS가 VPC와 타 클라우드를 전용 회선으로 잇는 AWS Interconnect를 프리뷰로 공개하고, 첫 파트너로 Google Cloud를 택했다. Azure 연동은 2026년 하반기 예정이다. 기존에 수 주~수 개월 걸리던 크로스클라우드 프라이빗 네트워킹을 수 분 내 구성할 수 있고, 대역폭·이중화가 기본 탑재된다.

> 💡 **왜 중요한가**: 하이퍼스케일러가 멀티클라우드를 공식 지원하는 전환점이며, 데이터 이동·AI 학습 파이프라인의 네트워크 병목과 Egress 비용 구조가 함께 바뀔 신호다.

### Anthropic, Claude Mythos 게이티드 프리뷰 시작 — Project Glasswing으로 50개 파트너 한정 공개

Anthropic이 4월 7일 차세대 모델 Claude Mythos를 Project Glasswing이라는 게이티드 프리뷰로 약 50개 파트너사에 공개했다. Opus 4.6 대비 "단계적 도약(step change)"으로 소개됐으며, 보안 취약점 탐지·추론·코딩에 특화된다. 앞서 미보호 DB에서 초안 블로그 글이 유출되며 외부에 먼저 알려진 모델이다.

> 💡 **왜 중요한가**: 프런티어 모델이 API 일반 공개 대신 계약 파트너 전용으로 풀리는 흐름이 굳어지고 있어, 개발자 접근성·벤치마크 검증 방식 모두 재편되고 있다.
