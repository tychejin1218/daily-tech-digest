### Cloudflare Agent Cloud 대규모 확장 — OpenAI GPT-5.4 직접 연동과 Dynamic Workers 공개

Cloudflare가 4월 13일 Agent Cloud에 OpenAI 프론티어 모델(GPT-5.4, Codex)을 네이티브 연동하고, AI 생성 코드를 샌드박스에서 격리 실행하는 Dynamic Workers와 Git 호환 저장소 Artifacts, Sandboxes GA를 함께 발표했다. Replicate 인수 후 통합한 단일 모델 카탈로그로 독점·오픈소스 모델을 하나의 창구에서 선택할 수 있다.

> 💡 **왜 중요한가**: 에이전트 런타임·스토리지·모델 카탈로그를 엣지 한 곳에 모으면서 백엔드 팀은 컨테이너·리전 관리 없이 수천만 저장소 규모의 에이전트 워크로드를 배포할 수 있게 됐다.

### Commvault, 에이전트 전용 백업 'AI Protect' 출시 — 에이전트 액션 Ctrl-Z로 롤백

Commvault가 AWS·Azure·GCP에서 동작하는 AI 에이전트를 자동 탐지·모니터링하고 잘못된 작업을 시점 복구(rollback)해 주는 AI Protect를 공개했다. 함께 발표된 Data Activate는 백업 데이터를 Apache Iceberg·Parquet 포맷으로 정제해 LLM 학습에 공급하고, AI Studio는 조직 맞춤 에이전트를 제작·배포한다.

> 💡 **왜 중요한가**: 에이전트가 프로덕션 DB를 잘못 건드렸을 때 '에이전트 단위 Ctrl-Z'가 가능해지면서, 백업/복구가 단순 DR을 넘어 에이전트 거버넌스의 실행 계층으로 재정의되고 있다.

### MiniMax M2.7 오픈소스 공개 — SWE-Pro 56.22%·Terminal Bench 2 57%로 코딩 SOTA 경신

중국 MiniMax가 4월 13일 공개한 M2.7은 실제 리포지토리 수정 능력을 측정하는 SWE-Pro에서 56.22%, 터미널 작업 벤치마크인 Terminal Bench 2에서 57.0%를 기록해 오픈소스 SOTA를 달성했다. Hugging Face와 Ollama Cloud에서 즉시 사용 가능하며 상용 라이선스가 허용된다.

> 💡 **왜 중요한가**: 독점 모델과의 코딩·터미널 성능 격차가 좁혀지면서, 온프레미스로 에이전트를 돌려야 하는 보안·규제 산업에서도 현실적인 대안이 생겼다.
