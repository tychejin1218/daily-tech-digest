# 2026-04-26 IT 뉴스 브리핑 — 백엔드 · 클라우드 · AI

### 1. DeepSeek, V4-Pro · V4-Flash 프리뷰 동시 공개 — 1M 컨텍스트 + 'Hybrid Attention'으로 프런티어 격차 좁히다 (4/24)
DeepSeek가 V4-Pro(1.6T total / 49B active)와 V4-Flash(284B total / 13B active) 두 MoE 모델을 동시에 프리뷰 공개했다. 새로운 'Hybrid Attention Architecture'로 100만 토큰 컨텍스트를 지원하면서도, V4-Flash는 100만 토큰당 입력 $0.14 / 출력 $0.28로 GPT-5.4 Nano · Gemini 3.1 Flash · Claude Haiku 4.5를 모두 가격에서 밀어붙였다.

> 💡 **왜 중요한가**: 오픈웨이트·저가 모델이 프런티어급 reasoning·agentic 벤치마크에 근접하면서, 멀티 프로바이더 라우팅 전략을 구축한 팀은 실질 추론 비용을 다시 한번 절반 수준으로 낮출 수 있는 카드가 생겼다.

### 2. OpenAI, ChatGPT 'Workspace Agents' 리서치 프리뷰 — Codex 기반 장기 실행 팀 에이전트 (4/22)
OpenAI가 ChatGPT Business · Enterprise · Edu 플랜에 'Workspace Agents'를 도입했다. Codex 런타임으로 클라우드에서 장기 실행되며, Slack · Google Drive · Microsoft 365 · Salesforce · Notion · Atlassian Rovo 등 엔터프라이즈 SaaS에 OAuth로 연결돼 팀 단위 권한·감사 로그 안에서 자동화된다. 5월 6일까지 무료, 이후 크레디트 기반 과금으로 전환된다.

> 💡 **왜 중요한가**: 'Custom GPTs'를 사실상 대체하는 후속 포지션으로, 개인용 챗봇 → 조직 자산화된 워크플로 에이전트로 진화하는 흐름이 OpenAI 진영에서 표준화되기 시작했다.

### 3. Snowflake, Snowflake Intelligence · Cortex Code 대규모 확장 — '에이전틱 엔터프라이즈 컨트롤 플레인' 천명 (4/21)
Snowflake가 Snowflake Intelligence에 자연어 워크플로(Skills)와 MCP 커넥터를 추가하고, Cortex Code를 AWS Glue · Databricks · 외부 PostgreSQL까지 확장해 데이터 이동 없이 거버넌스가 적용된 에이전트를 운영하도록 했다. 또한 Python·TypeScript용 Cortex Code Agent SDK를 공개해 외부 앱에 임베드할 수 있게 했고, Agent Communication Protocol로 타사 에이전트 프레임워크와 상호 운용된다.

> 💡 **왜 중요한가**: 데이터 웨어하우스가 단순 저장소를 넘어 '거버넌스가 내장된 에이전트 실행 플레인'으로 재포지셔닝되며, MCP·ACP가 사실상 엔터프라이즈 에이전트 인터컴 표준으로 굳어지고 있다.
